# ETL

## Goal
The goal of the project is build a ETL pipelie to extract data from different data sources, store, analyze and visulize the data. 

## Pipeline
### Data Extraction
The project will extract the data using New York Times and MarketStack api. Airflow 

### Data Ingestion
PostgresSQL will store the data

### Data Modelling
dtb is used to build data model
